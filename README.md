## Introduction

**PrismaQL** is a powerful tool for programmatic editing of **Prisma schema files** using a **SQL-like DSL**.  
It enables developers to **add, update, and remove models, fields, relations, and enums** while preserving **schema integrity** through an **AST-based engine**.

### Key Features:

- **DSL-powered Schema Editing** ‚Äì Modify Prisma schemas with a declarative and structured language.
- **Safe Modifications** ‚Äì Built-in **validation, commit tracking, and rollback capabilities**.
- **Query & Mutation System** ‚Äì Retrieve schema details or apply transformations programmatically.
- **Backup Mechanism** ‚Äì Schema changes are automatically **versioned and backed up**.
- **CLI & Programmatic API** ‚Äì Use it directly from the terminal or integrate it into your project.
